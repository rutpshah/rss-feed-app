
# RSS Feed App

Local environment setup:



## Run Locally

Clone the project

```bash
  git clone https://github.com/rutpshah/rss-feed-app
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Login with Google

```bash
  run localhost on port 3000 or 5473
```
